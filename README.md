# PBT-06 - Xây dựng giao diện với Bootstrap

Dự án này chứa các bài tập thực hành và câu trả lời lý thuyết về việc áp dụng Bootstrap framework để thiết kế giao diện web nhanh chóng, bao gồm trang đích (Landing Page) và bảng điều khiển (Dashboard).

## 📁 Cấu trúc thư mục

```text
PBT-06/
├── screenshots/
├── answers.md
├── bootstrap_dashboard.html
├── bootstrap_landing.html
└── README.md
```

- `screenshots/`: Chứa các hình ảnh kết quả của bài tập hiển thị giao diện trên trình duyệt.
- `answers.md`: Câu trả lời cho các câu hỏi lý thuyết liên quan đến hệ thống lưới (Grid System) và các component của Bootstrap.
- `bootstrap_dashboard.html`: Bài tập thực hành xây dựng giao diện trang quản trị (Admin Dashboard) sử dụng các thành phần của Bootstrap như Sidebar, Navbar, Cards và Tables.
- `bootstrap_landing.html`: Bài tập thực hành thiết kế trang đích (Landing Page) hiện đại, đáp ứng (responsive) với các phần Hero, Features, Footer,...
